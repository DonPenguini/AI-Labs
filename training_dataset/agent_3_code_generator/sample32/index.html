Langmuir Model

The Langmuir isotherm describes monolayer adsorption on a surface with a finite number of identical sites.

  • qmax: Maximum adsorption capacity
  • K: Equilibrium constant (affinity)
  • θ = q/qmax: Fractional surface coverage
