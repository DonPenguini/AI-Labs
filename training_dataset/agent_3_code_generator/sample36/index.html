Normal Modes

A double pendulum has two natural frequencies (eigenfrequencies) where the system oscillates in pure normal modes.

  • Mode 1: Both pendulums swing together (in-phase)
  • Mode 2: Pendulums swing oppositely (out-of-phase)
  • ω: Angular frequency (rad/s)
  • f: Frequency (Hz) = ω/(2π)
