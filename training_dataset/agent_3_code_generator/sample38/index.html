About Circular Orbits

In a circular orbit, the gravitational force provides exactly the centripetal force needed to maintain circular motion. The orbital speed depends only on the gravitational parameter (μ) and the orbital radius (r).

Astrodynamics Concepts

Two-Body Problem

Assumes only gravitational interaction between two bodies, ignoring all other forces and perturbations.

Circular Orbit

A special case where orbital eccentricity is zero and the distance from the central body remains constant.

Keplerian Motion

Orbital motion following Kepler's laws, valid for ideal two-body systems with inverse-square gravitational force.
