Formula Derivation

S = a + ar + ar² + ... + arn-1
rS = ar + ar² + ar³ + ... + arn
S - rS = a - arn
S(1 - r) = a(1 - rn)
S = a(1 - rn) / (1 - r)
Note: Formula undefined when r = 1 (arithmetic series)
